Multi-Account API Example 
# Add production Cloudflare account
curl -X POST "https://certmate.company.com/api/settings/dns-providers/cloudflare/accounts" \
  -H "Authorization: Bearer your_token" \
  -d '{"account_id": "production", "config": {"api_token": "prod_token"}}'

# Create certificate using specific account
curl -X POST "https://certmate.company.com/api/certificates/create" \
  -H "Authorization: Bearer your_token" \
  -d '{"domain": "api.company.com", "dns_provider": "cloudflare", "account_id": "production"}'

Quick Installation

Get CertMate running in under 5 minutes with Docker Compose

Docker Installation 
# Clone the repository
git clone https://github.com/fabriziosalmi/certmate.git
cd certmate

# Copy environment template
cp .env.example .env

# Edit .env with your DNS provider credentials
# Set API_BEARER_TOKEN and DNS provider settings

# Start all services
docker-compose up -d

# Check status
docker-compose ps
Python Installation 
# Create virtual environment
python3 -m venv certmate-env
source certmate-env/bin/activate

# Clone and install
git clone https://github.com/fabriziosalmi/certmate.git
cd certmate
pip install -r requirements.txt

# Set environment variables
export API_BEARER_TOKEN="your_token_here"
export CLOUDFLARE_TOKEN="your_cloudflare_token"

# Run the application
python app.py
Kubernetes Deployment 
# Create namespace
kubectl create namespace certmate

# Create secret for API token
kubectl create secret generic certmate-secrets \
  --from-literal=api-token=your_secure_token \
  -n certmate

# Apply deployment
kubectl apply -f k8s/deployment.yaml -n certmate

# Check status
kubectl get pods -n certmate

Powerful REST API

Complete programmatic control with comprehensive API endpoints

POST
/api/certificates/create
Create new SSL certificate with account selection
POST
/api/settings/dns-providers/{provider}/accounts
Add new DNS provider account
GET
/api/certificates
List all certificates with account info
GET
/{domain}/tls
Download certificate ZIP
API Example 
# Add multiple DNS accounts
curl -X POST "https://certmate.example.com/api/settings/dns-providers/cloudflare/accounts" \
  -H "Authorization: Bearer your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{"account_id": "production", "config": {"api_token": "prod_token"}}'

# Create certificate with specific account
curl -X POST "https://certmate.example.com/api/certificates/create" \
  -H "Authorization: Bearer your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{"domain": "api.company.com", "dns_provider": "cloudflare", "account_id": "production"}'

# Download certificate
curl -H "Authorization: Bearer your_api_token_here" \
     -o api.company.com-tls.zip \
     "https://certmate.example.com/api.company.com/tls"
