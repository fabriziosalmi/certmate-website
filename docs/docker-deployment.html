Docker Deployment

Production-ready containerized deployment

Quick Start

Multi-platform Docker deployment available on Docker Hub

Pull and Run

docker pull fabriziosalmi/certmate:latest
docker run -d --name certmate -p 8000:8000 --env-file .env fabriziosalmi/certmate:latest

Docker Compose

For production deployments, use Docker Compose for easier management:

# Clone repository
git clone https://github.com/fabriziosalmi/certmate.git
cd certmate

# Configure environment
cp .env.example .env
# Edit .env with your credentials

# Start services
docker-compose up -d

Supported Platforms

  • linux/amd64
  • linux/arm64
  • linux/arm/v7
