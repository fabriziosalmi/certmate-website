Backup & Recovery

Unified backup system and disaster recovery procedures

Unified Backup System

CertMate creates atomic backups of both settings and certificates ensuring data consistency.

Create Backup

curl -X POST "http://localhost:8000/api/backups/create" \
  -H "Authorization: Bearer your_token" \
  -d '{"reason": "manual_backup"}'

Restore from Backup

curl -X POST "http://localhost:8000/api/backups/restore/unified" \
  -H "Authorization: Bearer your_token" \
  -d '{"filename": "unified_backup_20260111_120000.zip"}'
