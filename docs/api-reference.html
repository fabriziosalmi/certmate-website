Certificate Management

GET/api/certificates

List all managed certificates

curl -H "Authorization: Bearer your_token" \
  http://localhost:8000/api/certificates
POST/api/certificates/create

Create a new certificate

Request Body:

Parameter Type Required Description
domain string ‚úÖ Domain name (e.g., example.com)
dns_provider string ‚ùå DNS provider (defaults to settings)
account_id string ‚ùå Specific account to use
domain_alias string ‚ùå Domain alias for DNS validation 
# Basic certificate creation
curl -X POST "http://localhost:8000/api/certificates/create" \
  -H "Authorization: Bearer your_token" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "example.com"
  }'

# With specific account
curl -X POST "http://localhost:8000/api/certificates/create" \
  -H "Authorization: Bearer your_token" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "staging.example.com",
    "dns_provider": "cloudflare",
    "account_id": "staging"
  }'

# With domain alias for centralized DNS validation
curl -X POST "http://localhost:8000/api/certificates/create" \
  -H "Authorization: Bearer your_token" \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "example.com",
    "dns_provider": "cloudflare",
    "domain_alias": "_acme-challenge.validation.example.org"
  }'

Python SDK Example

import requests
import zipfile
from pathlib import Path

class CertMateClient:
    def __init__(self, base_url, token):
        self.base_url = base_url.rstrip('/')
        self.headers = {"Authorization": f"Bearer {token}"}
    
    def create_certificate(self, domain, dns_provider=None, account_id=None):
        """Create a new certificate"""
        data = {"domain": domain}
        if dns_provider:
            data["dns_provider"] = dns_provider
        if account_id:
            data["account_id"] = account_id
        
        response = requests.post(
            f"{self.base_url}/api/certificates/create",
            headers=self.headers,
            json=data
        )
        return response.json()
    
    def download_certificate(self, domain, extract_to=None):
        """Download certificate as ZIP file"""
        response = requests.get(
            f"{self.base_url}/{domain}/tls",
            headers=self.headers
        )

Bash Automation Script

#!/bin/bash
set -euo pipefail

CERTMATE_URL="https://certmate.company.com"
API_TOKEN="${CERTMATE_TOKEN}"
DOMAIN="${1:-example.com}"
CERT_DIR="/etc/ssl/certs/${DOMAIN}"

# Download certificate
curl -f -H "Authorization: Bearer $API_TOKEN" \
     -o "${DOMAIN}-tls.zip" \
     "$CERTMATE_URL/$DOMAIN/tls"

# Extract to directory
mkdir -p "$CERT_DIR"
unzip -o "${DOMAIN}-tls.zip" -d "$CERT_DIR"
chmod 600 "$CERT_DIR"/*.pem

# Reload services
systemctl reload nginx
echo "Certificate updated for ${DOMAIN}"
